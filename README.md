## Environment Variables

To run this project, you need to add the following environment variables to your .env file.

| Name             | Type                                                                 |
| ----------------- | ------------------------------------------------------------------ |
| `dbHOST` | String |
| `dbPORT` | Int |
| `dbDATABASE` | String |
| `dbUSERNAME` | String |
| `dbPASSWORD` | String |
| `Key` | String |
